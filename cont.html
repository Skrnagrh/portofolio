Tentang Saya

Saya seorang lulusan Teknik Informatika dengan IPK 3.73 dari Universitas Serang Raya, memiliki pengalaman kerja di bidang administrasi dan layanan pelanggan. Berpengalaman sebagai Staff Administrasi di PT. Imam Putra Mandiri. Mahir dalam berbagai bahasa pemrograman dan alat pengembangan web seperti HTML, CSS, Java, PHP, Laravel, dan JavaScript, serta berpengalaman dalam menggunakan Git dan GitHub untuk kontrol versi. Memiliki keterampilan interpersonal yang kuat, termasuk kemampuan komunikasi, kerjasama tim, pemecahan masalah, dan manajemen waktu. Pengalaman organisasi di Paskibra menunjukkan kemampuan kepemimpinan, tanggung jawab, dan keterlibatan sosial yang baik.
